### Case 2: `eqW a a --> void`

This proof is straightforward. The ordinal measure `mu (eqW a a)` evaluates to `omega^(mu a + mu a + 9) + 1`, which is strictly greater than `0`, the measure of `void`.

```lean
-- eqW a a --> void
| _, _, Step.R_eq_refl a => by
    simp [mu]
    -- We need to show: 0 < omega^(mu a + mu a + 9) + 1
    -- This is true because omega to any power is positive, and adding 1 keeps it positive.
    apply add_pos_right
    apply opow_pos
    exact omega0_pos
```
