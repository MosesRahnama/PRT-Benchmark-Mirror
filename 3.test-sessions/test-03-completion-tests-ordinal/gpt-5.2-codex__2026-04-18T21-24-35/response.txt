### 2) `R_eq_refl` case
Goal:  
`0 < omega^(mu a + mu a + 9) + 1`

This is pure positivity. Skeleton:

```lean
| _, _, Step.R_eq_refl a => by
  simp [mu]
  -- show: 0 < omega0 ^ (mu a + mu a + 9) + 1
  -- use: add_pos_right + opow_pos
  apply Ordinal.pos_iff_ne_zero.mpr
  apply ne_of_gt
  -- omega^(...) > 0
  have h : 0 < omega0 ^ (mu a + mu a + (9:Ordinal)) :=
    opow_pos _ omega0_pos
  -- then omega^(...) + 1 > 0
  exact add_pos_right _ one_pos
```

If Lean needs a different lemma, you can replace with:

- `have h : (0:Ordinal) < omega0 ^ (...) := opow_pos _ omega0_pos`
- then `exact add_pos_right _ one_pos`

`mu (integrate (merge a b)) < mu (eqW a b)`

## Notes on helpful lemmas

These are the typical lemmas you’ll want to invoke or adapt:
