## 3) `Step.R_eq_diff` skeleton

\[
\omega^4(\mu(\mathrm{merge}\ a\ b)+1)+1 < \omega^{(\mu a + \mu b + 9)} + 1

1) reduce to `mu(integrate (merge a b)) < omega0 ^ (mu a + mu b + 9)` (then use `≤` into `+ 1`),
2) show `mu(merge a b) + 1 < omega0 ^ (mu a + mu b + 5)` (or any exponent that, after multiplying by `omega0^4`, lands under `omega0^(mu a+mu b+9)`),
3) rewrite `omega0 ^ (mu a + mu b + 9)` as `omega0^(4) * omega0^(mu a + mu b + 5)` using `opow_add`,
4) use monotonicity of left-multiplication by `omega0^4`,
5) absorb the final `+1` using that `omega0 ^ (...)` is limit.
