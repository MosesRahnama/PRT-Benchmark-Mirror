A workable skeleton is:

- `R_eq_refl`: same positivity pattern as `R_int_delta`.
- `R_rec_succ` and `R_eq_diff`: prove the left side is already `< ω^E`, then finish with `... ≤ ω^E + ...`.

Also, in the `R_rec_succ` comment block, the “LHS/RHS” labels are swapped; the goal is still correct.

### What to use in the `sorry` blocks
The recurring ingredient is:

- show some term is `< ω^E` by comparing exponents;
- then combine bounded summands using that `ω^E` is additively principal (or a limit ordinal with the usual closure property).

So the proof search should focus on lemmas about:

- monotonicity/strict monotonicity of `fun x => omega0 ^ x`,
- positivity of `omega0 ^ E`,
- closure of `ω^E` under addition of smaller ordinals.

If needed, package that closure once as a tiny local helper:
```lean
have hAP : IsAddPrincipal (omega0 ^ E) := by
  -- library lemma for powers of ω
  sorry
```
and then use it in both `R_rec_succ` and `R_eq_diff`.
