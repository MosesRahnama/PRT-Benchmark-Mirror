## Summary of Key Lemmas Needed

For a complete proof, you'll need these ordinal arithmetic facts:

1. **`opow_lt_opow_right`**: `a < b → ω^a < ω^b` (exponentiation is strictly increasing in the exponent)

2. **`opow_pos`**: `n > 0 → ω^n > 0`

3. **`le_mul_right`**: `a ≤ b * a` when `b ≥ 1`

4. **`max_le_add`**: `max a b ≤ a + b`

5. **Ordinal exponentiation absorption**: `ω^m * ω^n = ω^(m + n)` when `m, n < ω`

The critical insight is that the measure design ensures each rewrite rule's LHS has an ordinal tower that strictly dominates the RHS through careful exponent sizing (e.g., `omega^5` for `delta`, `+9` for `eqW`).
