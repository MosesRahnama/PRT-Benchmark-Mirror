**Key Calculation (Counterexample)**
Take `b = void`, `s = void`, and `n = delta void`.

1) Evaluate the left side of `kappa_rec_delta_step`:
- `kappa (recDelta void void (delta (delta void)))`
- By the `recDelta ... (.delta n)` clause and `kappa_delta`:
  - `kappa (recDelta void void (delta (delta void)))`
  - `= max (max (kappa void) (kappa void)) (kappa (delta void)) + 1`
  - `= max (max 0 0) (kappa void) + 1`
  - `= max 0 0 + 1`
  - `= 1`

2) Evaluate the right side:
- `kappa (recDelta void void (delta void)) + 1`
- Again by the `.delta n` clause:
  - `kappa (recDelta void void (delta void))`
  - `= max (max 0 0) (kappa void) + 1`
  - `= 1`
- So RHS `= 1 + 1 = 2`

Thus `kappa (recDelta void void (delta (delta void))) = 1` but `kappa (recDelta void void (delta void)) + 1 = 2`, so the claimed equality fails.
