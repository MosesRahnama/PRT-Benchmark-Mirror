  - If `n` is not a `delta` term (e.g., `void`, `integrate t`, etc.), then `kappa (recDelta b s n) = Nat.max (Nat.max (kappa b) (kappa s)) (kappa n)`.
  - If `n` is a `delta` term (e.g., `delta m`), then `kappa (recDelta b s n) = kappa (recDelta b s (delta m)) = Nat.max (Nat.max (kappa b) (kappa s)) (kappa m) + 1`.

- Left side: `Nat.max (Nat.max (kappa b) (kappa s)) (kappa (delta m)) + 1 = Nat.max (Nat.max (kappa b) (kappa s)) (kappa m) + 1`.

### Analysis of `kappa_rec_succ_drop`
This theorem claims: `kappa (app s (recDelta b s n)) < kappa (recDelta b s (delta n))` for all `b, s, n : Trace`, and relies on rewriting with `kappa_rec_delta_step`.

This requires `kappa s ≤ kappa (recDelta b s n)` and `kappa (recDelta b s n) < kappa (recDelta b s n) + 1` (the latter is trivially true).
