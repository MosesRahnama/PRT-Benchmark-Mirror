## Assessment: The Helper Strategy is **Unsound**

Both theorems fail when `n` is of the form `delta m`. Here's the analysis:

---

### Theorem 1: `kappa_rec_delta_step`

**Claim:** `kappa (recDelta b s (delta n)) = kappa (recDelta b s n) + 1`

**Counterexample:** Let `b = void`, `s = void`, `n = delta void`

| Expression | Calculation | Result |
|------------|-------------|--------|
| LHS: `kappa (recDelta void void (delta (delta void)))` | `max (max 0 0) (kappa (delta void)) + 1` = `max 0 0 + 1` | **1** |
| RHS: `kappa (recDelta void void (delta void)) + 1` | `(max (max 0 0) (kappa void) + 1) + 1` = `(0 + 1) + 1` | **2** |

**LHS ≠ RHS** — The theorem is false.

**Root cause:** The `kappa` function has a special case for `recDelta b s (delta n)` that adds +1, but when `n` itself is `delta m`, the recursive call `kappa (recDelta b s n)` *also* triggers the special case, yielding an extra +1 on the RHS.

---

### Theorem 2: `kappa_rec_succ_drop`

**Claim:** `kappa (app s (recDelta b s n)) < kappa (recDelta b s (delta n))`

**Same counterexample:** `b = void`, `s = void`, `n = delta void`

| Expression | Calculation | Result |
|------------|-------------|--------|
| LHS: `kappa (app void (recDelta void void (delta void)))` | `max 0 (kappa (recDelta void void (delta void)))` = `max 0 1` | **1** |
| RHS: `kappa (recDelta void void (delta (delta void)))` | `max (max 0 0) (kappa (delta void)) + 1` = `0 + 1` | **1** |

**LHS ≮ RHS** (they are equal) — The theorem is false.

---

### Diagnosis

The measure `kappa` "collapses" `delta` constructors via:
```lean
| .delta t => kappa t
```

This causes `kappa (delta (delta void)) = kappa (delta void) = kappa void = 0`, which undermines the intended structural decrease. The special `+1` in the `recDelta` case only applies to the *outermost* `delta`, not nested ones.

---

### Fix Options

1. **Add hypothesis:** Restrict `n` to not be of the form `delta _`:
   ```lean
   theorem kappa_rec_delta_step (b s n : Trace) (h : ∃ m, n = delta m → False) :
     kappa (recDelta b s (delta n)) = kappa (recDelta b s n) + 1
   ```

2. **Redesign `kappa`:** Make it count `delta` depth properly, e.g.:
   ```lean
   | .delta t => kappa t + 1
   ```
