### Counterexample for `kappa_rec_delta_step`
The theorem claims: `kappa (recDelta b s (delta n)) = kappa (recDelta b s n) + 1`.

- Let `n = delta void`.
- Left side: `kappa (recDelta b s (delta (delta void)))`.
  - Third arg is `delta (delta void)`, matching the `+1` case in `kappa`.
  - `kappa (delta (delta void)) = kappa (delta void) = kappa void = 0`.
  - So, `kappa = max(max(0, 0), 0) + 1 = 1`.
- Right side: `kappa (recDelta b s (delta void)) + 1`.
  - Third arg is `delta void`, matching the `+1` case in `kappa`.
  - `kappa (delta void) = kappa void = 0`.
  - So, `kappa (recDelta b s (delta void)) = max(max(0, 0), 0) + 1 = 1`.
  - Right side: `1 + 1 = 2`.
- `1 ≠ 2`, so the equality fails.

This holds only if `n` is not of the form `delta m` (e.g., it works if `n = void`, yielding `1 = 0 + 1`).

### Counterexample for `kappa_rec_succ_drop`

- Let `n = delta void`.
