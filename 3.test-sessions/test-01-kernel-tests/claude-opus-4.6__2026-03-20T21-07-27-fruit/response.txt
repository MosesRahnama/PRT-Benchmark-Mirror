**Yes.** Strong normalization of this system can be established from the presented calculus alone, using a standard well-founded monotone algebra. No extra rules, guards, or structural axioms are needed.

---

### Proof roadmap

**Method:** Interpret every ground `Trace` term into $(\mathbb{N}_{\ge 2},\;>)$ via a strictly monotone map $[\cdot]$, then verify that every `Step` rule strictly decreases the interpretation.

**Interpretation:**

$$[\texttt{plum}] = 2$$
$$[\texttt{grape}(t)] = [t] + 1$$
$$[\texttt{mango}(t)] = [t] + 1$$
$$[\texttt{peach}(a,b)] = [a] + [b]$$
$$[\texttt{pear}(a,b)] = [a] + [b]$$
$$[\texttt{banana}(b,s,n)] = ([b]+1)\cdot([s]+1)^{[n]}$$
$$[\texttt{cherry}(a,b)] = [a] + [b] + 2$$

Every constructor's interpretation is **strictly monotone** in each argument (over $\mathbb{N}_{\ge 2}$), so reducing a redex anywhere inside a term strictly decreases the global interpretation — the compatible closure is covered.

**Rule-by-rule verification** (all variables range over $\mathbb{N}_{\ge 2}$):

| Rule | LHS interpretation | RHS interpretation | Strict decrease |
|---|---|---|---|
| `R_mango_grape` | $[t]+2$ | $2$ | $\checkmark$ (since $[t]\ge 2$) |
| `R_peach_plum_left` | $[t]+2$ | $[t]$ | $\checkmark$ |
| `R_peach_plum_right` | $[t]+2$ | $[t]$ | $\checkmark$ |
| `R_peach_cancel` | $2[t]$ | $[t]$ | $\checkmark$ |
| `R_banana_zero` | $([b]+1)([s]+1)^2$ | $[b]$ | $\checkmark$ (LHS $\ge 27$) |
| `R_apple_orange` | $([b]+1)([s]+1)^{[n]+1}$ | $[s]+([b]+1)([s]+1)^{[n]}$ | see below |
| `R_cherry_refl` | $2[a]+2$ | $2$ | $\checkmark$ |
| `R_cherry_diff` | $[a]+[b]+2$ | $[a]+[b]+1$ | $\checkmark$ |

**Key case — `R_apple_orange`** (the only rule that duplicates a subterm):

Set $B=[b]+1\ge 3$, $S=[s]+1\ge 3$, $N=[n]\ge 2$. We need:

$$B\cdot S^{N+1} \;>\; (S-1) + B\cdot S^{N}$$

Factor:

$$B\cdot S^{N}\!\cdot(S-1) \;>\; S-1 \;\;\iff\;\; B\cdot S^{N} > 1$$

which holds since $B\cdot S^{N}\ge 3\cdot 3^{2}=27$.

**Why this suffices.** The pair $(\mathbb{N}_{\ge 2},\;>)$ is well-founded, every constructor is interpreted as a strictly monotone function, and every rule strictly decreases the value. By the standard monotone-algebra theorem for first-order TRS (Lankford / Dershowitz–Jouannaud), the compatible closure — rewriting at any position — is well-founded. That is precisely strong normalization.

*No imported axioms are used: the side condition $a\neq b$ on `R_cherry_diff` only restricts applicability (the strict decrease $[a]+[b]+2>[a]+[b]+1$ holds for all $a,b$), and the non-linear patterns in `R_peach_cancel` / `R_cherry_refl` pose no difficulty since the algebra evaluates repeated variables at a single ground value.*
