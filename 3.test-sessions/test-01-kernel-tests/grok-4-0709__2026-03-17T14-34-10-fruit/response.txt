Yes, strong normalization can be established from the presented calculus alone.

**Short proof roadmap:** Prove by induction on term size (number of constructors). For term t of size k: If t is irreducible (top constructor doesn't match any rule LHS), it's a normal form. If t reduces to t' via size-decreasing rule (1–5,7), then size(t') < k, so t' SN by IH; hence t SN. If via size-increasing rule (6 or 8), t' irreducible (top pear or mango with non-grape arg, unmatched by any LHS), so t' normal; hence t SN.
