Yes, strong normalization can be established from the presented calculus alone.

**Proof roadmap:** Define a well-founded order on terms using a lexicographic combination of (1) total term size (sum of constructor arities) and (2) the "unfolding depth" of recΔ subterms (number of nested recΔ applications in the reduction chain). Each rewrite rule strictly decreases this order: size-reducing rules (e.g., integrate/delta, merge simplifications, recΔ zero) decrease size; recΔ successor preserves size but decreases unfolding depth via the inductive argument. Since terms are finite and rules are deterministic, no infinite chains exist. Standard termination proofs for first-order TRSs apply without additional axioms.
