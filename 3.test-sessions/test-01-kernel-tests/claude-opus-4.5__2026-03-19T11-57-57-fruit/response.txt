**Proof Roadmap (ordinal interpretation):**

Define `⟦·⟧ : Trace → Ord`:
- `⟦plum⟧ = 0`
- `⟦grape(t)⟧ = ⟦t⟧ + 1`
- `⟦mango(t)⟧ = ⟦t⟧ + 1`
- `⟦peach(a, b)⟧ = max(⟦a⟧, ⟦b⟧) + 1`
- `⟦pear(a, b)⟧ = ⟦a⟧ + ⟦b⟧ + 1`
- `⟦banana(b, s, n)⟧ = ω^⟦n⟧ · (⟦s⟧ + 1) + ⟦b⟧`
- `⟦cherry(a, b)⟧ = ω^(max(⟦a⟧, ⟦b⟧) + 1) + ⟦a⟧ + ⟦b⟧`

**Verification of strict decrease:**

| Rule | LHS → RHS check |
|------|-----------------|
| `R_apple_orange` | `ω^(n+1)·(s+1) + b > s + ω^n·(s+1) + b + 1` ✓ (ordinal: `ω·α > α + k` for finite k) |
