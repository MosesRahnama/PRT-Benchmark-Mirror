There‚Äôs no rule that *creates* a new `delta` (so ‚Äúrecursion depth‚Äù can‚Äôt be regenerated), no rule that turns `void` back into anything, and no rule that recreates `eqW`/`merge`/`integrate` redex-shapes from their contracta in a way that can cycle. ## Why a na√Øve ‚Äúsize decreases‚Äù argument fails (but termination still looks provable)

A simple ‚Äúterm size strictly decreases‚Äù measure **does not** work globally because:

- `R_rec_succ : recŒî b s (delta n) ‚Üí app s (recŒî b s n)`

duplicates `s` (one copy at the `app` head plus one still inside the recursive call), so raw size can increase.

That blocks the most elementary proof method, but it does **not** block standard termination methods like recursive path ordering.
