1. **Define a well‑founded measure** on traces.

   |t| = \text{number of constructors in }t

   and give an extra weight to the third argument of `recΔ` that counts the number of nested `δ`‑constructors:

   \[
   w(\text{recΔ}\;b\;s\;u) = |b|+|s|+|u|+\#\delta(u)

   where `\#δ(u)` is the length of the `δ`‑chain in `u` (0 for `void`).  
   The overall measure of a pair `(t₁,t₂)` on which a `Step` is applied is the sum of the measures of the two traces.

2. **Show each rule strictly decreases the measure**  

   * `R_int_delta` : `integrate (δ t) → void`.  
     `|integrate (δ t)| = 1 + 1 + |t|` while `|void| = 0`. Decrease ≥ 2.

   * `R_merge_void_left/right` : `merge void t → t` (or symmetrically).  
     `|merge void t| = 1 + 0 + |t|` → `|t|`. Decrease = 1.

   * `R_merge_cancel` : `merge t t → t`.  
     `|merge t t| = 1 + 2·|t|` → `|t|`. Decrease ≥ |t| + 1 > 0.

   * `R_rec_zero` : `recΔ b s void → b`.  
     `|recΔ b s void| = 1 + |b| + |s| + 0` → `|b|`. Decrease = |s| + 1 > 0.

   * `R_rec_succ` : `recΔ b s (δ n) → app s (recΔ b s n)`.  
     The left‑hand side contributes one extra `δ`. The right‑hand side replaces that `δ` by an `app` plus a smaller recursive call `recΔ b s n`. Hence  

     \[
     w(\text{recΔ }b s (δ n)) = 1+|b|+|s|+(\#δ(n)+1)
     \]
     \[
     w(\text{app }s (\text{recΔ }b s n)) = 1+|s|+w(\text{recΔ }b s n)
     \]
     The difference is exactly 1 (the removed `δ`). So the measure drops.

   * `R_eq_refl` : `eqW a a → void`.  
     `|eqW a a| = 1 + 2·|a|` → `0`. Decrease ≥ 2·|a| + 1.

   * `R_eq_diff` : `eqW a b → integrate (merge a b)`.  
     The right‑hand side replaces `eqW` (weight 1 + |a| + |b|) by `integrate (merge a b)` whose weight is `1 + (1 + |a| + |b|) = 2 + |a| + |b|`.

Consequently every reduction sequence that starts with `R_eq_diff` must eventually apply `R_int_delta` (or a merge rule) and strictly decreases the measure.
