## Proof Roadmap via Polynomial Interpretation

Define the following polynomial interpretation over $\mathbb{N}$:

$$
[\![Z]\!] = 0,\quad [\![S(z)]\!] = z + 1
$$
$$
[\![F(x,y,z)]\!] = x + yz + z + 1
$$
$$
[\![G(x,y)]\!] = x + y
$$

**Verify rule 1** — $F(x, y, Z) \to x$:

$$
[\![F(x,y,0)]\!] = x + y \cdot 0 + 0 + 1 = x + 1 > x \quad \checkmark
$$

**Verify rule 2** — $F(x, y, S(n)) \to G(y, F(x, y, n))$:

$$
[\![F(x, y, n+1)]\!] = x + y(n+1) + (n+1) + 1 = x + yn + y + n + 2
$$
$$
[\![G(y,\, F(x,y,n))]\!] = y + (x + yn + n + 1) = x + yn + y + n + 1
$$
$$
\text{Difference} = 1 > 0 \quad \checkmark
$$

Both rules are strictly decreasing. The interpretation uses only polynomials with non-negative coefficients (so strict monotonicity holds throughout), and the well-ordering of $\mathbb{N}$ closes the argument.
