The proposed lexicographic termination measure `(phase t, cost t)` is not sound. For several rewrite rules (including `R_merge_void_left`, `R_merge_void_right`, `R_merge_cancel`, and `R_rec_zero`), the measure can increase when the target term has phase 1 while the source has phase 0. For example, in `merge t t → t` where `t = recDelta b s (delta n)`, the source measure is `(0, 1 + 2 \times \text{cost } t)` and the target is `(1, \text{cost } t)`, but $(0, \cdot) < (1, \cdot)$ in the lexicographic order on $\mathbb{N} \times \mathbb{N}$. Similar issues occur in other simplifying rules that can "expose" a subterm with phase 1.
