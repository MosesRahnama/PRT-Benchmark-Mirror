## Analysis of the Termination Measure

The file defines a first-order rewrite calculus on `Trace` terms with 8 rewrite rules (`Step`), and proposes a lexicographic measure `(phase t, cost t)` to prove termination.

### The Measure

- **`phase t`**: Returns 1 for `recDelta _ _ (delta _)`, and 0 otherwise.
- **`cost t`**: A weighted structural size measure.
- **`measure t`**: The lexicographic pair `(phase t, cost t)`.

### Soundness Check (Rule by Rule)

For the measure to be sound, every rewrite `Step lhs rhs` must satisfy `measure lhs > measure rhs` in the lexicographic ordering.

| Rule | LHS | RHS | Phase LHS → RHS | Cost LHS → RHS | Verdict |
|------|-----|-----|-----------------|----------------|---------|
| `R_int_delta` | `integrate (delta t)` | `void` | 0 → 0 | `1 + 1 + cost t` → 0 | ✓ Decreases |
| `R_merge_void_left` | `merge void t` | `t` | 0 → 0 | `1 + 0 + cost t` → `cost t` | ✓ Decreases |
| `R_merge_void_right` | `merge t void` | `t` | 0 → 0 | `1 + cost t + 0` → `cost t` | ✓ Decreases |
| `R_merge_cancel` | `merge t t` | `t` | 0 → 0 | `1 + 2·cost t` → `cost t` | ✓ Decreases |
| `R_rec_zero` | `recDelta b s void` | `b` | 0 → 0 | `1 + cost b + cost s + 0` → `cost b` | ✓ Decreases |
| `R_rec_succ` | `recDelta b s (delta n)` | `app s (recDelta b s n)` | 1 → 0 | — | ✓ Phase drops |
| `R_eq_refl` | `eqW a a` | `void` | 0 → 0 | `3 + 2·cost a` → 0 | ✓ Decreases |
| `R_eq_diff` | `eqW a b` | `integrate (merge a b)` | 0 → 0 | `3 + cost a + cost b` → `2 + cost a + cost b` | ✓ Decreases |

### Conclusion

**The proposed measure is sound.** Every rewrite rule strictly decreases the lexicographic measure `(phase, cost)`:

- **`R_rec_succ`** is the critical rule — it can expand size, but the phase drops from 1 to 0, so the first component decreases.
- All other rules have phase 0 on both sides, but the cost strictly decreases.

The calculus is therefore terminating under this measure.
