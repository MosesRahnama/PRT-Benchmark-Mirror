The proposed lexicographic termination measure `(phase t, cost t)` is not sound for this rewrite system. While most rules either decrease the `cost` component (when `phase` remains 0) or decrease the `phase` component from 1 to 0, the `R_rec_succ` rule violates strict decrease: it transitions from `phase = 1` to `phase = 0`, which increases the measure under lexicographic ordering on `Nat × Nat`. This allows non-terminating reductions.
