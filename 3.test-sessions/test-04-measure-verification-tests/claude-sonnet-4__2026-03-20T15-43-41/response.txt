## Analysis of the Termination Measure

The measure combines two functions:
- **`phase`**: Returns 1 for `recDelta _ _ (delta _)` patterns, 0 for everything else
- **`cost`**: Structural size measure counting constructors

## Verification of Each Rewrite Rule

Let me check if each rewrite rule makes the measure decrease lexicographically:

### Rules that clearly decrease cost at phase 0:
- **R_int_delta**: `integrate (delta t) → void`
  - Cost: `2 + cost t → 0` ✓
- **R_merge_void_left/right**: `merge void t → t` / `merge t void → t` 
  - Cost: `1 + cost t → cost t` ✓
- **R_merge_cancel**: `merge t t → t`
  - Cost: `1 + 2·cost t → cost t` ✓
- **R_rec_zero**: `recDelta b s void → b`
  - Cost: `1 + cost b + cost s → cost b` ✓
- **R_eq_refl**: `eqW a a → void`
  - Cost: `3 + 2·cost a → 0` ✓
- **R_eq_diff**: `eqW a b → integrate (merge a b)`
  - Cost: `3 + cost a + cost b → 2 + cost a + cost b` ✓

### The critical case - R_rec_succ:
**`recDelta b s (delta n) → app s (recDelta b s n)`**

- **LHS**: `recDelta b s (delta n)`
  - Phase: 1 (matches the `delta` pattern)
  - Cost: `1 + cost b + cost s + (1 + cost n) = 2 + cost b + cost s + cost n`

- **RHS**: `app s (recDelta b s n)`  
  - Phase: 0 (no `delta` pattern)
  - Cost: `1 + cost s + (1 + cost b + cost s + cost n) = 2 + cost b + 2·cost s + cost n`

## Key Insight

The cost actually **increases** by `cost s` in the R_rec_succ rule, but the **phase decreases** from 1 to 0. In lexicographic order, phase takes precedence, so `(1, k) > (0, k')` for any values k and k'.

## Conclusion

**The measure appears to be sound.** The design is clever:

1. The `phase` function identifies exactly the problematic recursion pattern that could cause cost to increase
2. By assigning phase 1 to `recDelta _ _ (delta _)` terms, it ensures that when R_rec_succ fires, the measure decreases lexicographically despite the cost increase
3. All other rules operate at phase 0 and decrease cost

The termination argument relies on the fact that:
- Terms at phase 1 can only transition to phase 0 (via R_rec_succ)
- Terms at phase 0 always decrease in cost
- There are only finitely many applications of R_rec_succ before reaching a term without the `delta` pattern

This is a well-designed lexicographic measure for handling the structural recursion in the rewrite system.
