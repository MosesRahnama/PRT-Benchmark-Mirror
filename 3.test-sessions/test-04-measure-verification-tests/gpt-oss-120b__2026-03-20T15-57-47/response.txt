**Assessment of the lexicographic termination measure**

The file defines a *measure* on terms of the rewrite calculus:

```lean
def measure (t : Trace) : Nat × Nat := (phase t, cost t)
```

where  

* `phase t` is `1` **only** for terms of the form `recDelta _ _ (delta _)`; otherwise it is `0`.  
* `cost t` is a natural‑number weight that counts the constructors (with a larger weight for `eqW`).

A rewrite step `Step l r` is *sound* with respect to this measure if  

```
measure l  >  measure r          (lexicographic order)
```

i.e. either  

1. `phase l > phase r`, or  
2. `phase l = phase r` **and** `cost l > cost r`.

Below we examine each constructor of `Step`.

| Rule | Left‑hand side (`l`) | Right‑hand side (`r`) | `phase` comparison | `cost` comparison | Verdict |
|------|----------------------|-----------------------|--------------------|-------------------|---------|
| `R_int_delta` | `integrate (delta t)` | `void` | `0 = 0` | `2 + cost t  > 0` | **decreases** |
| `R_merge_void_left` | `merge void t` | `t` | `0 = phase t` (may be `0` or `1`) | `1 + cost t  >?  cost t` (true) | **fails when `phase t = 1`** |
| `R_merge_void_right` | `merge t void` | `t` | same as above | same | **fails when `phase t = 1`** |
| `R_merge_cancel` | `merge t t` | `t` | `0 = phase t` (may be `1`) | `1 + 2·cost t  >  cost t` (true) | **fails when `phase t = 1`** |
| `R_rec_zero` | `recDelta b s void` | `b` | `0 = phase b` (may be `1`) | `1 + cost b + cost s  >  cost b` (true) | **fails when `phase b = 1`** |
| `R_rec_succ` | `recDelta b s (delta n)` | `app s (recDelta b s n)` | `1 > 0` | – | **decreases** |
| `R_eq_refl` | `eqW a a` | `void` | `0 = 0` | `3 + 2·cost a  > 0` | **decreases** |
| `R_eq_diff` | `eqW a b` | `integrate (merge a b)` | `0 = 0` | `3 + cost a + cost b  >  2 + cost a + cost b` | **decreases** |

### Why the measure is **not sound**

The problematic rules are the three *merge* rules and `R_rec_zero`.  
In each of those cases the left‑hand side has `phase = 0`, while the right‑hand side can have `phase = 1` (e.g. when `t` or `b` is itself a `recDelta … (delta …)`).  

Lexicographically, `(0, …) < (1, …)`. Hence the measure **increases** for those steps, violating the required monotonic decrease.

Because at least one rewrite rule does not strictly decrease the measure, the lexicographic pair `(phase, cost)` is **not a sound termination measure** for this rewrite calculus. A sound measure would need to ensure that every `Step` reduces the first component whenever the second component could increase, which is not the case here.
