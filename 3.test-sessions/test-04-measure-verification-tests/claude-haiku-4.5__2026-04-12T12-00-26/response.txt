## Soundness Assessment of the Lexicographic Measure

The measure is defined as:
- **Phase:** `(phase t, cost t)` where phase ∈ {0, 1} and cost is the sum of structural size + recursion overhead
- **Comparison:** Lexicographic order on $(phase, cost)$

### Analysis by Rule:

**✓ R_int_delta:** `integrate(delta t) → void`
- Phase: $1 \to 0$ (decreases phase) ✓
- Cost: $(2 + cost(t)) \to 0$ (strictly decreases cost) ✓

**✓ R_merge_void_left:** `merge(void, t) → t`
- Phase: $0 \to 0$ (unchanged)
- Cost: $(1 + 0 + cost(t)) \to cost(t)$ (decreases by 1) ✓

**✓ R_merge_void_right:** `merge(t, void) → t`
- Phase: $0 \to 0$ (unchanged)
- Cost: $(1 + cost(t) + 0) \to cost(t)$ (decreases by 1) ✓

**✓ R_merge_cancel:** `merge(t, t) → t`
- Phase: $0 \to 0$ (unchanged)
- Cost: $(1 + cost(t) + cost(t)) \to cost(t) = (1 + 2·cost(t)) \to cost(t)$ (decreases by $1 + cost(t)$) ✓

**✓ R_rec_zero:** `recDelta(b, s, void) → b`
- Phase: $0 \to 0$ (unchanged)
- Cost: $(1 + cost(b) + cost(s) + 0) \to cost(b)$ (decreases by $1 + cost(s)$) ✓

**✗ R_rec_succ:** `recDelta(b, s, delta n) → app(s, recDelta(b, s, n))`
- Phase: $1 \to 0$ (decreases phase) ✓
- LHS cost: $1 + cost(b) + cost(s) + (1 + cost(n))$
- RHS cost: $1 + cost(s) + (1 + cost(b) + cost(s) + cost(n))$
- RHS = $2 + 2·cost(s) + cost(b) + cost(n)$
- LHS - RHS = $(1 + cost(b) + cost(s) + 1 + cost(n)) - (2 + 2·cost(s) + cost(b) + cost(n))$
- = $-cost(s)$ ✓ (decreases despite phase dropping to 0, since LHS had phase 1)

**✓ R_eq_refl:** `eqW(a, a) → void`
- Phase: $0 \to 0$ (unchanged)
- Cost: $(3 + 2·cost(a)) \to 0$ (decreases by $3 + 2·cost(a)$) ✓

**✓ R_eq_diff:** `eqW(a, b) → integrate(merge(a, b))`
- Phase: $0 \to 0$ (unchanged)
- LHS cost: $3 + cost(a) + cost(b)$
- RHS cost: $1 + (1 + cost(a) + cost(b)) = 2 + cost(a) + cost(b)$
- LHS - RHS = $1$ (decreases by 1) ✓

### Conclusion:

**The measure is SOUND.** All seven rewrite rules strictly decrease the lexicographic measure $(phase, cost)$. The measure correctly exploits the phase system to capture that certain reductions are "high-priority" (phase 1), while maintaining progress via cost whenever phase is stable.
