**Assessing measure soundness and lexicographic order**

The measure under discussion combines two components—phase and cost—assumed to be ordered lexicographically with phase as the first element and cost as the second. This means the overall measure is expected to decrease if either the phase decreases or the phase stays the same and the cost decreases.

However, a concrete counterexample shows this measure isn't sound as currently defined. Consider the term `t := recDelta void void (delta void)`, which has `phase t = 1`. Applying the rewrite rule `R_merge_void_left` or similar rules like `R_merge_void_right`, `R_merge_cancel`, or `R_rec_zero` can produce a left-hand side term (merging void with `t`) of phase 0. The measure comparison goes from `(0, 1 + cost t)` on the left to `(1, cost t)` on the right after rewriting.

In lexicographic order, since phase is the primary component and 0 < 1, `(0, 1 + cost t)` is actually less than `(1, cost t)`. So the measure increases after rewriting, violating soundness because we expect it to decrease or stay the same.

If the intention was a different ordering, such as reversing the lexicographic order to make phase descending (so 1 < 0), or using the order `(phase, cost)` but interpreting phase as a simple 0/1 with 1 > 0, the measure could potentially be sound. But as currently written and assuming a standard lexicographic order with phase ascending, the measure doesn't guarantee reduction through these rules.
