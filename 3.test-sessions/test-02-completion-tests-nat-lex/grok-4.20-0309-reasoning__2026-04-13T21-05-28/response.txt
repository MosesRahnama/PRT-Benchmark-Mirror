**Proof skeleton for the remaining cases in `measure_decreases`**

Replace each `sorry` with the corresponding block below. These skeletons follow the exact style and helper lemmas (`drop_left`/`drop_right`, `simp` lemmas, `omega`) already used in the three proved merge cases. They handle the lexicographic order on `(kappa, mu)` by either:

* Proving strict decrease on `kappa` (then `drop_left`), or
* Proving `kappa` ties and strict decrease on `mu` (then `drop_right`).

All cases are exhaustive; the `by_cases` branches are only needed where `kappa` may tie.

-- integrate (delta t) --> void
| _, _, Step.R_int_delta t => by
    have hmu : mu void < mu (integrate (delta t)) := by simp [mu]; omega
    by_cases h : kappa t = 0
    · have hk : kappa void = kappa (integrate (delta t)) := by simp [kappa, h]
      exact drop_right hmu hk
    · have hk : kappa void < kappa (integrate (delta t)) := by simp [kappa, h]; omega
      exact drop_left hk

-- recDelta b s void --> b
| _, _, Step.R_rec_zero b s => by
    have hmu : mu b < mu (recDelta b s void) := by simp [mu]; omega
    by_cases h : kappa b < kappa (recDelta b s void)
    · exact drop_left h
    · have hk : kappa b = kappa (recDelta b s void) := by
        simp [kappa] at *; omega
      exact drop_right hmu hk

-- recDelta b s (delta n) --> app s (recDelta b s n)
| _, _, Step.R_rec_succ b s n => by
    have hk : kappa (app s (recDelta b s n)) < kappa (recDelta b s (delta n)) := by
      simp only [kappa]
      rw [kappa_rec_delta]  -- uses the provided @[simp] lemma
      apply Nat.lt_succ_of_le
      exact Nat.le_max_left _ _
    exact drop_left hk

-- eqW a a --> void
| _, _, Step.R_eq_refl a => by

    by_cases h : kappa a = 0
    · have hk : kappa void = kappa (eqW a a) := by simp [kappa, h]
      exact drop_right hmu hk

-- eqW a b --> integrate (merge a b)
| _, _, Step.R_eq_diff a b => by
