- Or showing `kappa` remains equal and `mu` decreases strictly (right component), giving `LexOrder` decrease via `drop_right`.

The proofs rely on the `@[simp]` lemmas for `kappa` and `mu`, along with basic arithmetic (e.g., `omega` for inequalities). ### Case: `Step.R_int_delta t` (integrate (delta t) --> void)
```
| _, _, Step.R_int_delta t => by
  -- kappa decreases from kappa t to 0 (unless kappa t = 0, in which case it ties)
  -- mu decreases from 1 + mu t to 0
  cases Nat.lt_or_eq 0 (kappa t) with
  | inl hk => exact drop_left hk  -- kappa decreases strictly
  | inr hk => 
    have hmu : mu void < mu (integrate (delta t)) := by simp [mu]; omega
    exact drop_right hmu hk  -- kappa ties, mu decreases
```
