.

Use the same pattern as the finished `merge` cases: prove either a strict `mu` drop with equal `kappa`, or a strict `kappa` drop.

### `integrate (delta t) --> void`
`kappa` drops from `kappa t` to `0`, so split on `t = void` vs `t ≠ void` or simply prove by arithmetic on `mu` with tied `kappa` when `kappa t = 0`.

Skeleton:
```lean
| _, _, Step.R_int_delta t => by
    by_cases hk0 : kappa t = 0
    · have hmu : mu void < mu (integrate (delta t)) := by
        simp [mu]
        omega
      have hk : kappa void = kappa (integrate (delta t)) := by
        simp [kappa, hk0]
      exact drop_right hmu hk
    · have hk : kappa void < kappa (integrate (delta t)) := by
        simp [kappa]
        omega
      exact drop_left hk
```

A shorter version may also work if `omega` can discharge `0 < kappa t` from `hk0`.

---

### `recDelta b s void --> b`
Here `kappa b ≤ kappa (recDelta b s void)` by definition, so the natural route is `kappa` tie/drop plus `mu` drop. Since equality is not guaranteed, easiest is to branch.

Skeleton:
```lean
| _, _, Step.R_rec_zero b s => by
    by_cases hlt : kappa b < kappa (recDelta b s void)
    · exact drop_left hlt
    · have hk : kappa b = kappa (recDelta b s void) := by
        simp [kappa] at hlt ⊢
        omega
      have hmu : mu b < mu (recDelta b s void) := by
        simp [mu]
        omega
      exact drop_right hmu hk
```

If preferred, first prove
```lean
have hle : kappa b ≤ kappa (recDelta b s void) := by simp [kappa]
```
then derive equality from `not_lt_of_ge`.

---

### `recDelta b s (delta n) --> app s (recDelta b s n)`
This is the key recursive case. The left side has
$$
kappa(recDelta\ b\ s\ (\delta n)) =
\max(\max(\kappa b,\kappa s),\kappa n)+1
$$
while the right side has
$$
kappa(app\ s\ (recDelta\ b\ s\ n))
= \max(\kappa s,\max(\max(\kappa b,\kappa s),\kappa n)).
$$
So `kappa` drops strictly by the extra `+1`.

Skeleton:
```lean
| _, _, Step.R_rec_succ b s n => by
    have hk : kappa (app s (recDelta b s n)) < kappa (recDelta b s (delta n)) := by
      simp [kappa, Nat.lt_succ_self]
      set m : Nat := Nat.max (Nat.max (kappa b) (kappa s)) (kappa n)
      have hs : kappa s ≤ m := by
        dsimp [m]
        exact le_trans (Nat.le_max_right _ _) (Nat.le_max_left _ _)
      have happ : Nat.max (kappa s) m = m := by
        exact Nat.max_eq_right hs
      rw [happ]
      omega
    exact drop_left hk
```

A terser version may be:
```lean
    have hk : kappa (app s (recDelta b s n)) < kappa (recDelta b s (delta n)) := by
      simp [kappa]
      set m : Nat := Nat.max (Nat.max (kappa b) (kappa s)) (kappa n)
      have hs : kappa s ≤ m := by
        omega
      rw [Nat.max_eq_right hs]
      omega
```

### `eqW a a --> void`
Usually `kappa` may or may not drop, so the safe route is again branch on whether `kappa a = 0`.

Skeleton:
```lean
| _, _, Step.R_eq_refl a => by
    by_cases hk0 : kappa a = 0
    · have hmu : mu void < mu (eqW a a) := by
        simp [mu]
        omega
      have hk : kappa void = kappa (eqW a a) := by
        simp [kappa, hk0]
      exact drop_right hmu hk
    · have hk : kappa void < kappa (eqW a a) := by
        simp [kappa]
        have : 0 < kappa a := by omega
        simpa using this
      exact drop_left hk
```

Because `kappa (eqW a a) = kappa a`, this case is very similar to `R_int_delta`.

---

### `eqW a b --> integrate (merge a b)`
Here `kappa` is unchanged:
$$
\max(\kappa a,\kappa b)
$$
on both sides. So prove strict `mu` decrease:
- source: `4 + mu a + mu b`
- target: `1 + (2 + mu a + mu b) = 3 + mu a + mu b`

Skeleton:
```lean
| _, _, Step.R_eq_diff a b => by
    have hmu : mu (integrate (merge a b)) < mu (eqW a b) := by
      simp [mu]
      omega
    have hk : kappa (integrate (merge a b)) = kappa (eqW a b) := by
      simp [kappa]
    exact drop_right hmu hk
```

## Compact version

If you want a near drop-in completion block, this is the cleanest skeleton:

```lean
| _, _, Step.R_int_delta t => by
    by_cases hk0 : kappa t = 0
    · have hmu : mu void < mu (integrate (delta t)) := by
        simp [mu]
        omega
      have hk : kappa void = kappa (integrate (delta t)) := by
        simp [kappa, hk0]
      exact drop_right hmu hk
    · have hk : kappa void < kappa (integrate (delta t)) := by
        simp [kappa]
        omega
      exact drop_left hk

| _, _, Step.R_rec_zero b s => by
    by_cases hlt : kappa b < kappa (recDelta b s void)
    · exact drop_left hlt
    · have hk : kappa b = kappa (recDelta b s void) := by
        simp [kappa] at hlt ⊢
        omega
      have hmu : mu b < mu (recDelta b s void) := by
        simp [mu]
        omega
      exact drop_right hmu hk

| _, _, Step.R_rec_succ b s n => by
    have hk : kappa (app s (recDelta b s n)) < kappa (recDelta b s (delta n)) := by
      set m : Nat := Nat.max (Nat.max (kappa b) (kappa s)) (kappa n)
      have hs : kappa s ≤ m := by
        dsimp [m]
        exact le_trans (Nat.le_max_right _ _) (Nat.le_max_left _ _)
      have happ : Nat.max (kappa s) m = m := by
        exact Nat.max_eq_right hs
      simp [kappa, m, happ]
    exact drop_left hk

| _, _, Step.R_eq_refl a => by
    by_cases hk0 : kappa a = 0
    · have hmu : mu void < mu (eqW a a) := by
        simp [mu]
        omega
      have hk : kappa void = kappa (eqW a a) := by
        simp [kappa, hk0]
      exact drop_right hmu hk
    · have hk : kappa void < kappa (eqW a a) := by
        simp [kappa]
        omega
      exact drop_left hk

| _, _, Step.R_eq_diff a b => by
    have hmu : mu (integrate (merge a b)) < mu (eqW a b) := by
      simp [mu]
      omega
    have hk : kappa (integrate (merge a b)) = kappa (eqW a b) := by
      simp [kappa]
    exact drop_right hmu hk
```

One likely refinement: in `R_rec_succ`, `simp [kappa, m, happ]` may need a final `omega`.
