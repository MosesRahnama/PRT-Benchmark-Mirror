Here is the assessment of the three candidates. A measure $\mu$ is valid iff for every rule $\ell \to r$ in `Step`, $\mu(\ell) > \mu(r)$.

## Rule-by-rule analysis

Let $b, s, n$ stand for arbitrary sub-terms with $\mu$-values $B, S, N$ respectively (all $\ge 0$).

| **mu2** | $2+(1+T) = 3+T$ | $0$ | $+3+T$ ✓ |
| **mu3** | $2+(1+T) = 3+T$ | $0$ | $+3+T$ ✓ |

### R_rec_succ: `recDelta b s (delta n)` → `app s (recDelta b s n)`

LHS:
- **mu1**: $1+B+S+(1+N) = 2+B+S+N$
- **mu2**: $5+B+S+(1+N) = 6+B+S+N$
- **mu3**: $1+B+S+(1+N) = 2+B+S+N$

RHS = `app s (recDelta b s n)`:
- **mu1**: $1+S+(1+B+S+N) = 2+B+2S+N$
- **mu2**: $1+S+(5+B+S+N) = 6+B+2S+N$
- **mu3**: $2+S+(1+B+S+N) = 3+B+2S+N$

Δ = LHS − RHS:
- **mu1**: $(2+B+S+N)-(2+B+2S+N) = -S$. For $S=0$ (e.g. $s = \texttt{void}$) this is $0$. **Not strictly decreasing. FAIL.**
- **mu2**: $(6+B+S+N)-(6+B+2S+N) = -S$. Same issue. **FAIL.**
- **mu3**: $(2+B+S+N)-(3+B+2S+N) = -1-S \le -1$. **Always strictly decreasing. PASS.**

### R_eq_refl: `eqW a a` → `void`

### R_eq_diff: `eqW a b` → `integrate (merge a b)`

LHS / RHS:
- **mu1**: $(3+A+B)$ vs $(1+(1+A+B)) = 2+A+B$. Δ = $1$ ✓
- **mu2**: $(5+A+B)$ vs $(2+(2+A+B)) = 4+A+B$. Δ = $1$ ✓
- **mu3**: $(4+A+B)$ vs $(2+(1+A+B)) = 3+A+B$. Δ = $1$ ✓

## Verdict

| Candidate | R_rec_succ | All others | Valid? |
|---|---|---|---|
| **mu1** | **FAILS** ($\Delta = -S$, collapses when $s=\texttt{void}$) | pass | **No** |
| **mu2** | **FAILS** ($\Delta = -S$, same collapse) | pass | **No** |
| **mu3** | $\Delta = -1-S \le -1$, always decreasing | pass | **Yes** |

`mu3` is the only valid termination measure. The failure of `mu1` and `mu2` on `R_rec_succ` is structural: both assign weight $1$ to `app`, so the outer `app` wrapper on the RHS costs only $1$, while the recursive call reproduces the full $s$-cost. `mu3` fixes this by assigning weight $2$ to `app` instead of $1$, so the outer constructor always contributes a surplus that dominates the extra copy of $S$.
