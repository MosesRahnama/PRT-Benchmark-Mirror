**Assessment of the three termination measures (`mu1`, `mu2`, `mu3`)**

The rewrite calculus is given by the inductive relation `Step : Trace ‚Üí Trace ‚Üí Prop`.  
For a measure‚ÄØ`Œº : Trace ‚Üí Nat` to be a *valid termination measure* it must satisfy

\[
\forall\, a\,b,\; Step\;a\;b \;\Longrightarrow\; Œº(a) > Œº(b)
\]

(i.e. the measure strictly decreases on every rewrite step).

Below we evaluate each rule of `Step` against the three candidate measures.

| Rule | Left‚Äëhand side (LHS) | Right‚Äëhand side (RHS) | `Œº1(LHS)` vs `Œº1(RHS)` | `Œº2(LHS)` vs `Œº2(RHS)` | `Œº3(LHS)` vs `Œº3(RHS)` |
|------|----------------------|-----------------------|------------------------|------------------------|------------------------|
| **R_int_delta** | `integrate (delta t)` | `void` | `2 + Œº1 t  > 0` | `3 + Œº2 t  > 0` | `3 + Œº3 t  > 0` |
| **R_merge_void_left** | `merge void t` | `t` | `1 + Œº1 t  > Œº1 t` | `2 + Œº2 t  > Œº2 t` | `1 + Œº3 t  > Œº3 t` |
| **R_merge_void_right** | symmetric | symmetric | same as above | same as above | same as above |
| **R_merge_cancel** | `merge t t` | `t` | `1 + 2¬∑Œº1 t  > Œº1 t` | `2 + 2¬∑Œº2 t  > Œº2 t` | `1 + 2¬∑Œº3 t  > Œº3 t` |
| **R_rec_zero** | `recDelta b s void` | `b` | `1 + Œº1 b + Œº1 s  > Œº1 b` | `5 + Œº2 b + Œº2 s  > Œº2 b` | `1 + Œº3 b + Œº3 s  > Œº3 b` |
| **R_rec_succ** | `recDelta b s (delta n)` | `app s (recDelta b s n)` | **`2 + Œº1 b + Œº1 s + Œº1 n`** vs **`2 + Œº1 b + 2¬∑Œº1 s + Œº1 n`** ‚Üí *fails* (right‚Äëhand side larger by `Œº1 s`) | **`6 + Œº2 b + Œº2 s + Œº2 n`** vs **`6 + Œº2 b + 2¬∑Œº2 s + Œº2 n`** ‚Üí *fails* (right‚Äëhand side larger by `Œº2 s`) | **`2 + Œº3 b + Œº3 s + Œº3 n`** vs **`3 + Œº3 b + 2¬∑Œº3 s + Œº3 n`** ‚Üí *fails* (right‚Äëhand side larger by `1 + Œº3 s`) |
| **R_eq_refl** | `eqW a a` | `void` | `3 + 2¬∑Œº1 a  > 0` | `5 + 2¬∑Œº2 a  > 0` | `4 + 2¬∑Œº3 a  > 0` |
| **R_eq_diff** | `eqW a b` | `integrate (merge a b)` | `3 + Œº1 a + Œº1 b  > 2 + Œº1 a + Œº1 b` (decrease by‚ÄØ1) | `5 + Œº2 a + Œº2 b  > 4 + Œº2 a + Œº2 b` (decrease by‚ÄØ1) | `4 + Œº3 a + Œº3 b  > 3 + Œº3 a + Œº3 b` (decrease by‚ÄØ1) |

### Summary of findings

* **All three measures (`mu1`, `mu2`, `mu3`) decrease on every rule except `R_rec_succ`.**  
  For `R_rec_succ` the right‚Äëhand side (`app s (recDelta b s n)`) has a *larger* measure than the left‚Äëhand side (`recDelta b s (delta n)`). The increase is proportional to the measure of `s` (and, for `mu3`, an additional constant‚ÄØ1). Consequently the required strict decrease condition is violated.

* Because a single rule fails to satisfy the decreasing‚Äëmeasure condition, **none of the three candidates constitutes a valid global termination measure** for the rewrite system.

### Conclusion

- `mu1`, `mu2`, and `mu3` are *partial* measures: they correctly rank most rewrites, but they all break on the recursive step `R_rec_succ`.  
- To obtain a sound termination argument, a different measure (or a combination of measures) must be devised that strictly decreases for `R_rec_succ` as well‚Äîe.g., a lexicographic pair that gives priority to the ‚Äúdepth‚Äù of the `delta`‚Äënesting or that counts the number of pending `delta` constructors separately.  

**Therefore, none of the three proposed termination measures is sufficient on its own.**
