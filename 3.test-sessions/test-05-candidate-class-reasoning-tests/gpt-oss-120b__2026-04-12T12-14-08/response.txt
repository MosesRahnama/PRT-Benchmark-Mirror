### Overall conclusion
None of the three proposed measures (`mu1`, `mu2`, `mu3`) constitute a valid termination measure for the rewrite calculus defined in
.  

* `mu1` and `mu2` are *non‑increasing* (they can stay equal) on the `R_rec_succ` rule, which is insufficient for a well‑founded decreasing metric.  
* `mu3` even **increases** on that rule, making it outright unsuitable.

A correct termination measure must assign a strictly smaller natural number to the right‑hand side of **every** rewrite step, in particular to the `R_rec_succ` rule. A new candidate should therefore give a larger weight to the `app` constructor (or a smaller weight to `recDelta`) so that the inequality  

\[
\mu(\text{recDelta } b\; s\; (\delta n)) \;>\; \mu(\text{app } s\; (\text{recDelta } b\; s\; n))
\]

holds for all traces `b, s, n`.
