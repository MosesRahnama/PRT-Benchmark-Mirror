Primitive Recursor Test (PRT) Benchmark: a benchmark of large language model behavior on nine formal-methods termination-reasoning task families. PRT is set at the minimal first-order primitive recursor F(x,y,Z) -> x, F(x,y,S(n)) -> G(y,F(x,y,n)), evaluated at two tiers (the schema kernel and KO7, an eight-rule first-order calculus) across 27 frontier models and 1,188 sessions on nine tasks. Each session is graded on three axes: termination, witness adequacy, and rule-extracted boundary admissibility, with an externally-checkable answer key fixed by archived TTT2/CeTA certifications and a companion mechanized Lean 4 theorem stack.
